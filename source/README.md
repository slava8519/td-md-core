# Источник истины (диссертация)

- **`time_decomposition.docx`** — кандидатская диссертация Андреева В.В. (2007). Канонический источник: физика, метод, параметры, формулы, рисунки, таблицы. **При любом сомнении сверяйся с ней.**
- **`time_decomposition.txt`** — автоматический текстовый экстракт (~144k символов) для быстрого поиска (`grep`). Внимание: рисунки, таблицы и формулы в оригинале — изображения, в `.txt` они отсутствуют. За ними — в `.docx`.

Этот каталог не входит в набор инструкций для сборки (`docs/`), но является первоисточником, из которого они выведены.
